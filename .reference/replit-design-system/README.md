# 🎯 REPLIT DESIGN SYSTEM REFERENCE

This folder contains the complete extracted design system from the Replit reference project.

## 📚 CONTENTS
- `css-system.md` - Complete CSS variables and utilities
- `component-structure.md` - Component architecture
- `tailwind-config.md` - Animation and theme configuration
- `AssetCard.tsx` - 3D flip card component
- `GlassCard.tsx` - Glass morphism component

## 🚀 USAGE
Reference these files when implementing the Replit design system in new projects.

- Deep space color scheme (#0A0F2F background)
- Electric cyan accents (#00D1FF)
- Neon purple secondary (#7B61FF)
- Glass morphism with backdrop blur
- 3D flip card animations
- Particle background system
- Hover lift effects
- Neon glow utilities
